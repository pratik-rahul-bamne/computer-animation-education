A modern, interactive educational website exploring the fascinating world of computer animation, VFX, and interactive media.

### 📱 Fully Responsive Design
- Mobile-first approach with seamless desktop scaling
- Optimized for tablets, phones, and large displays
- Touch-friendly interface elements

### 🎨 Modern Visual Design
- Stunning gradient color scheme (blues, purples, whites)
- Smooth animations and transitions
- Particle background effects
- Glowing and hover effects
- Custom CSS animations

#### 🏠 Hero Section
- Animated particle background
- Gradient text effects

## 🎯 Educational Content Covered

### Animation Fundamentals
- **2D Animation**: Traditional frame-by-frame techniques
- **3D Animation**: Computer-generated three-dimensional motion
- **Stop Motion**: Physical object manipulation
- **Motion Capture**: Real-world movement recording

### Advanced Techniques
- **Keyframe Animation**: Defining key positions for smooth interpolation
- **Procedural Animation**: Algorithm-driven automated movement
- **Animation Sequencing**: Organizing complex animation workflows
- **Motion Control**: Various methods for controlling object movement

### Visual Effects
- **VFX (Visual Effects)**: Digital post-production effects
- **SFX (Special Effects)**: Practical on-set effects
- **Morphing**: Smooth transformation between shapes/images
- **Warping**: Geometric distortion techniques

### Emerging Technologies
- **Virtual Reality (VR)**: Immersive 360° digital environments
- **Augmented Reality (AR)**: Digital overlay on real world
- **Applications**: Gaming, education, training, entertainment
